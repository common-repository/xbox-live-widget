A widget that displays gamercard or tree different types of avatar full body, icon large or icon small. It uses the official Xbox live URLs.

== Description ==

Xbox Live Widget adds an easy way to show gamertags on the sidebars, there are four types of gamertags available in this version:

*   Full gamertag from Xbox.com
*   Full body image of your avatar
*   Large icon of your avatar
*   Small icon of your avatar

The four options are implemented with official xbox live urls.

As with many applications using third party data sources, the plugin functions will depend on gamertag and avatar availability from xbox.com

== Installation ==

1. Upload `bm-xbox-live-widget.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to `Appearance` section and select `Widgets`
4. Drag and drop the `Xbox live` widget to the desired sidebar

I was searching for a widget with my full body avatar that were linked with my Xbox Live account but I only could find the Gamertag so I decided to create a widget with all the available Gamertag/Avatars.
